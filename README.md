# bearworks (MkDocs Site)

このリポジトリは、学習まとめサイト（MkDocs）のソースコードを管理しています。
`main` ブランチにプッシュすると、Netlifyの自動ビルドによりサイトが公開されます。

## ローカルでの開発・プレビュー方法

ローカル環境で記事を執筆しながら画面を確認したい場合は、以下のコマンドを実行してください。

```bash
uv run mkdocs serve
```

※ このコマンドを使うと、静的な `site/` フォルダを生成しなくてもローカルサーバーが立ち上がり、ファイルの変更がリアルタイムでプレビュー画面に反映されます。
